Projects

Check out our software projects below to find brief descriptions, point people, and some links to resources and more information.

The STC Project

Lead: Andrew Milligan

The STC project encapsulates all of the embedded software development for tracker and heliostat controls. The current state of the art is an embedded, on-the-metal program written in C to be run on a Microchip Pic32 to control Solaflect trackers and heliostats.

The Gateway Project

Lead: Andrew Milligan

The Stratosphere Gateway project encapsulates all of the software that runs on a Solaflect Network Gateway to facilitate network communication between a STC and the Cirrus Servers.

View Details >>

Solaflect VPN

Lead: Andrew Milligan

The Solaflect VPN was set up as a backend debugging infrastructure tool to allow remote terminal access to Stratosphere Network Gateways. The VPN is administered by a VPN server running OpenVPN. Solaflect developed a suite of tools to help technicias use the VPN for debugging purposes. The tools are mostly written in Bash.

View Details >>

The mkgw Project

Lead: Andrew Milligan

The mkgw package contains tools to provision a new Gateway. This includes setting it up on the VPN, installing the Stratosphere package, installation of all dependencies, and more.
